Le design atomique

Avec le découpage atomique on oublie l'intégration page par page. Le découpage en css se fait ainsi:

  • Les atomes

    Le plus petit composant. Il s'agit de skinner les titres, paragraphes, boutons, inputs... Il s'agit d'éléments uniques au niveau du CSS. Dans la structure d'Adfab, ces éléments seront dans le dossier /layout/. A ce niveau les éléments ont des information de style (couleur, police, padding...) mais pas d'information de positionnement.

  • Les molécules

    Un regroupement d'atomes. Dans le dossier /components/, ce sont les éléments qu'on retrouve dans les blocs. Les pictos de réseaux sociaux, un formulaire de newsletter.

  • Les organismes

    Chaque molécule est disposée sur un ou plusieurs blocs et positionnée comme il faut dessus. Le style des blocs se trouvant dans le dossier /blocs/.

  • Les templates/pages

    Les blocs sont affichés les uns à la suite des autres sur des templates qui donneront lieu aux pages du site une fois dynamisés. Normalement il y a très peu de CSS au niveau des templates, l'essentiel étant déjà fait aux niveaux précédents. Mais si besoin le fichier LESS/SASS dans le dossier /pages/ doit avoir le même nom que le template, afin de s'y retrouver facilement.

De nombreuses informations sur ce sujet sont disponibles sur http://atomicdesign.bradfrost.com/.

Découpage des maquettes

Dès la réception des maquettes une étape important est d'identifier sur l'ensemble du site les éléments récurrents qui constitueront les composants et blocs du site.

Identifier les titres, boutons, liens... qui seront autant d'atomes, molécules et organismes. Sur certaines pages ou blocs ces composants peuvent avoir de légères variations (police plus grosse sur la home page, couleur différente sur une page ou un bloc), mais ça restera le même composant s'il a la même signification sur la page.

La home page est généralement la page la plus spécifique du site, avec des blocs uniques ou stylisés différemment, mais c'est aussi la plupart du temps la première page intégrée. En faisant un découpage avec l'intégralité des pages du site, il faut penser au bloc générique qu'on trouve sur le plus de page, et le surcharger pour la home page, et non l'inverse, pour avoir un code plus simple.

Méthodologie

  1. Sous photoshop ou tout autre logiciel d'édition graphique, mettez les jpeg des maquettes côte à côte sur un même document.
  2. Créez un groupe dans lequel vous mettrez vous calques pour identifier les éléments, avec une opacité réduite pour voir l'original derrière
  3. Commencez par identifier les éléments atomiques: boutons, titres, sous-titres... qui reviennent sur chaque composants/blocs/pages. Créez un calque puis sélectionnés tous les éléments d'un même type.
  4. Une fois tous les atomes d'un type sélectionné, remplissez la sélection avec une couleur.
  5. Recommencez l'opération pour chaque élément atomique en créant un nouveau calque avec une nouvelle couleur de fond à chaque fois.
  6. Faites un nouveau groupe, de nouveaux calques et la même méthode pour les composants, puis les blocs.

Uns fois ce travail terminé, presque tous les élément de l'ensemble des créas doivent être surlignés. Vous devriez voir facilement tout un tas de soucis dans les créas de cette manière (tous les titres d'une couleur sauf sur une page, un bouton plus gros sur un template...). N'hésitez pas à demander validation au client si c'est normal ou si vous uniformisez dans le code.
